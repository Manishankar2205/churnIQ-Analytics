# ChurnIQ Dashboard

Customer Churn Prediction using Flask + XGBoost

### Setup

1. `pip install -r requirements.txt`
2. `python train_model.py`
3. `python app.py`
4. Open http://127.0.0.1:5000
5. Login: admin / admin

### Features

- Dark theme + Glassmorphism UI
- 4 KPI cards + 3 Plotly charts
- ML prediction with Gauge chart
- Flask-Login authentication
